## Link
```bash
//cdn.jsdelivr.net/gh/user/repo/file.js
```
```bash
//cdn.jsdelivr.net/gh/user/repo@version/file.js
```
```bash
//cdn.jsdelivr.net/gh/user/repo@latest/file.js
```
```bash
//cdn.jsdelivr.net/gh/user/repo@master/file.js
```
## 刷新缓存
```
//cdn.jsdelivr.net/
```
替换成
```
//purge.jsdelivr.net/
```
即可实时刷新
